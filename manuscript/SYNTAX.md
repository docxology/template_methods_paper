# Manuscript Syntax Reference (template_methods_paper)

Project-specific overlay on the canonical
[`docs/guides/manuscript-semantics.md`](../../../../docs/guides/manuscript-semantics.md)
— read that file first; this file documents the **template_methods_paper**
figure registry, table registry, and `{{TOKEN}}` table.

## Citation Syntax (Pandoc)

```markdown
[@bpl2026]
[@kahn1962topological; @merkle1987digital]
@fowler2010dsl makes the case that...
```

All citation keys must exist in [`references.bib`](references.bib). Pandoc with
`--natbib` converts `[@key]` to the right LaTeX cite command automatically;
**never** write raw `\cite{}` in Markdown.

## Figure References

```markdown
![Steps per example method](../output/figures/step_counts.png){#fig:step_counts width=80%}

[@fig:step_counts] shows...
```

- Images must exist in `output/figures/` at render time (run
  `scripts/methods_analysis.py` first).
- Use `width=` or `height=` to control sizing.
- Captions are self-contained — they name the figure's generator.

### Figure label registry

|---|---|---|
| `{#fig:step_counts}` | `output/figures/step_counts.png` | `scripts/methods_analysis.py::run_methods_analysis` (matplotlib bar chart) |

## `{{TOKEN}}` Reference

Every numeric or run-dependent value in prose is a `{{TOKEN}}`, generated by
`src/manuscript_variables.py::generate_variables` and injected by
`scripts/z_generate_manuscript_variables.py` via
`infrastructure.rendering.manuscript_injection.write_resolved_manuscript_tree`.
A token used in any `manuscript/*.md` file that the generator does not emit
fails `tests/test_manuscript_variables.py`.

| Token | Source |
|---|---|
| `{{CONFIG_VERSION}}` | `manuscript/config.yaml` `paper.version` |
| `{{CONFIG_FIRST_AUTHOR}}` | `manuscript/config.yaml` `authors[0].name` |
| `{{CONFIG_KEYWORDS}}` | `manuscript/config.yaml` `keywords` (joined) |
| `{{DSL_UNIT_COUNT}}` | `len(known_units())` |
| `{{DSL_STEP_KIND_COUNT}}` | `len(StepKind)` |
| `{{DSL_TARGET_COUNT}}` | `len(Target)` |
| `{{DSL_GATE_COUNT}}` | Fixed at 4 (structural, semantic, plan, target) |
| `{{EXAMPLE_METHOD_COUNT}}` | `len(all_example_methods())` |
| `{{DETERMINISM_CHECK}}` | `Yes`/`No` — live re-compilation hash-equality check |
| `{{PBS_STEP_COUNT}}` / `{{PBS_PLAN_HASH}}` / `{{PBS_TARGET}}` | `output/data/compiled_plans.json` (`PBSPreparation`) |
| `{{CALIBRATION_STEP_COUNT}}` / `{{CALIBRATION_PLAN_HASH}}` / `{{CALIBRATION_TARGET}}` | `output/data/compiled_plans.json` (`SensorCalibrationSweep`) |
| `{{TOTAL_GATES_RUN}}` / `{{TOTAL_GATES_PASSED}}` | `output/reports/gate_report.json` |
| `{{TRUST_CHAIN_LENGTH}}` / `{{TRUST_CHAIN_VERIFIED}}` | `output/reports/trust_chain_report.json` |
| `{{ARTIFACT_FIGURES}}` / `{{ARTIFACT_DATA_FILES}}` / `{{ARTIFACT_REPORTS}}` / `{{ARTIFACT_TOTAL}}` | Counted files under `output/{figures,data,reports}` |
| `{{CONFIG_HASH}}` | SHA-256 of `manuscript/config.yaml` (first 16 hex chars) |
| `{{GENERATION_TIMESTAMP}}` | Build time (honors `SOURCE_DATE_EPOCH`) |
| `{{PYTHON_VERSION}}` / `{{PLATFORM}}` | `platform.python_version()` / `platform.system() + platform.machine()` |

## Preamble Injection

[`preamble.md`](preamble.md) contains the LaTeX packages Pandoc consumes via
`infrastructure.rendering.latex_utils`. Do not duplicate package imports already
in the infrastructure renderer.

## Section Numbering

Files are assembled in lexicographic order by
`infrastructure/rendering/pdf_renderer.py`. The `00_` prefix renders the abstract
first; `99_` renders references last.

## Prose Conventions

- No "In summary" or "In conclusion" at section ends (RASP standard).
- Use active voice.
- Reference code with explicit paths: `src/methods_dsl/compiler.py`, not "the
  compiler module".
